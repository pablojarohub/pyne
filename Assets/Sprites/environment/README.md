# Environment Sprites

Place your environment sprites here:

- `road_horizontal.png` - Horizontal road segment
- `road_vertical.png` - Vertical road segment
- `road_intersection.png` - Road intersection
- `sidewalk.png` - Sidewalk tile
- `building_1.png` - Generic building 1
- `building_2.png` - Generic building 2
- `building_3.png` - Generic building 3
- `tree.png` - Decorative tree
- `bench.png` - Decorative bench
- `streetlamp.png` - Street lamp

## Style Guide

- Top-down or side-view perspective
- Thick black outlines
- Bright but balanced colors
- Tileable textures for roads and sidewalks
- Building sizes: 128x128 to 256x256 pixels
