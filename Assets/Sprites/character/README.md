# Character Sprites

Place your character sprites here:

- `character_idle.png` - Standing pose
- `character_ride_1.png` - Riding frame 1
- `character_ride_2.png` - Riding frame 2
- `character_happy.png` - Happy expression
- `character_tired.png` - Tired expression
- `character_stressed.png` - Stressed expression

## Style Guide

- Big, chubby body
- Orange outfit + orange delivery backpack
- Thick black outlines
- Bright, saturated colors
- Cartoon / arcade style
- Size: approximately 128x128 pixels per frame
