# Bike Sprites

Place your bicycle sprites here:

- `bike_frame.png` - Main bike body
- `bike_wheel.png` - Wheel (can be rotated in code)
- `bike_pedal_1.png` - Pedal frame 1
- `bike_pedal_2.png` - Pedal frame 2

## Style Guide

- Simple bicycle design
- Thick black outlines
- Bright colors (red, blue, or orange)
- Size: approximately 96x64 pixels
