# UI Assets

Place your UI sprites here:

- `arrow.png` - Directional arrow indicator
- `panel_bg.png` - Panel background (rounded rectangle)
- `button_normal.png` - Button default state
- `button_hover.png` - Button hover state
- `button_pressed.png` - Button pressed state
- `timer_circle.png` - Circular timer fill
- `icon_burger.png` - Burger icon for Burger Place
- `icon_kebab.png` - Kebab icon for Kebab King
- `icon_market.png` - Market icon for Market Store
- `logo.png` - Game logo for main menu

## Style Guide

- Cartoon / arcade aesthetic
- Thick outlines on icons
- Bright, readable colors
- Consistent corner radius on panels and buttons
